# Retail Sales Analysis

## Dataset Includes
- Transaction date
- Customer ID
- Transaction amount
- Service category
- Product type
- City & state
- Payment method (credit/debit)

## Analysis Performed
- Total and average sales by service and product
- Customer purchase behavior
- State and city performance
- Monthly and quarterly sales trends
- Payment method analysis
- Identification of high-performing and underperforming categories

## Tools Used
- Python
- Pandas
- Datetime operations
- GroupBy and aggregation functions

## Purpose
To understand sales patterns, customer behavior, geographic performance, and seasonal trends for better business decision-making.
